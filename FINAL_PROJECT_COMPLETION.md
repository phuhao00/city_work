│   │   ├── applications/   # 申请管理组件
│   │   └── search/         # 搜索功能组件
│   ├── services/           # API服务层
│   ├── store/              # Redux状态管理
│   ├── navigation/         # 路由导航
│   └── theme/              # 主题和样式
```

### 后端架构 (Node.js)
```
backend/
├── src/
│   ├── auth/               # 身份认证模块
│   ├── users/              # 用户管理
│   ├── companies/          # 企业管理
│   ├── jobs/               # 职位管理
│   ├── messaging/          # 消息系统
│   ├── search/             # 搜索引擎
│   └── common/             # 共享模块
